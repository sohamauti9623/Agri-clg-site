# Shri Vitthal College of Agriculture Website

Premium university-style website built with Next.js 15, TypeScript, Tailwind CSS, and Framer Motion.

## Commands

```bash
npm install
npm run dev
npm run build
npm run typecheck
```

## Preview URL

Every pull request automatically gets a Vercel preview deployment through `.github/workflows/vercel-preview.yml`. Add the following repository secrets before opening or updating a PR:

- `VERCEL_TOKEN`
- `VERCEL_ORG_ID`
- `VERCEL_PROJECT_ID`

After the workflow finishes, it comments the live preview URL on the pull request and writes it to the GitHub Actions step summary.
